# Professor Diniz

Site institucional do Professor João Diniz Marcello.

## Estrutura

- `index.html`
- `style.css`
- `script.js`
- `assets/professor.jpg` para substituir pela foto oficial

## Publicação no GitHub Pages

1. Envie os arquivos para o repositório.
2. Vá em Settings > Pages.
3. Em Source, selecione Deploy from a branch.
4. Selecione branch `main` e pasta `/root`.
5. Salve.

Depois configure o domínio `professordiniz.com.br` no GitHub Pages.
